Background Images Applied to a Page

Free Background Images

The are a number of site on the Internet where you can download free background images for web pages. You can also create such images using applications such as Photoshop or Paint Shop Pro.

Adding a background image

Inserting the Image

  1. save the image into your css folder
  2. click the "Page Properties" button
  3. click the "Appearance (CSS)" category if needed
  4. click the "Browse" button next to the background image selection box, browse to the image and select it.
  5. Click "OK" to close the "Page Properties" settings.

Setting How an Image Tiles

Using the repeat drop down menu under the image selection box:

  • choose no-repeat to display the image once
  • choose repeat-x to have the image tile in a horizontal line across the top
  • choose repeat-y to have the image tile in a vertical line along the left margin
  • pick repeat or leave the box without a selection to have the image tile across the entire page.
