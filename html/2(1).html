Assignment 2

In this assignment, we started to learn how to format text using simple <p> and <h> tags inside of the page properties of Dreamweaver. In the Page Properties of DW, we can change the color of text and headings without having to do it individually to each line of text.

 

Then we formatted the text to read like a website, and linked it to another page within Dreamweaver, all without having to hand-code anything.
